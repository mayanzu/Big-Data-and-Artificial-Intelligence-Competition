## 练习1：Pandas 数据处理（第一周）

用 Python 从仿真平台获取数据，进行数据清洗和分析。

### 1.1 获取数据

```python
import pandas as pd
import requests

# 从仿真平台获取 500 条数据
response = requests.get("http://163.7.1.176:8889/api/data?n=500")
data = response.json()
df = pd.DataFrame(data)

print(df.shape)   # 应该显示 (500, 11)
print(df.columns) # 查看所有列名
print(df.head())  # 查看前5行
```

### 1.2 数据清洗

```python
# 练习：检查缺失值
print(df.isnull().sum())

# 练习：删除 price 为空的行
df_clean = df.dropna(subset=['price'])
print(f"清理后: {len(df_clean)} 行")

# 练习：按品类统计
print(df.groupby('category')['price'].mean().sort_values(ascending=False))
```

### 1.3 数据分析

```python
# 练习：统计各事件类型数量
print(df['event_type'].value_counts())

# 练习：按设备统计
print(df['device'].value_counts(normalize=True))

# 练习：筛选购买记录
buys = df[df['event_type'] == 'buy']
print(f"购买记录: {len(buys)} 条")
print(buys['category'].value_counts())
```

部署大数据平台后，用以下命令验证数据流。


```sql

    user_id STRING, event_type STRING, session_id STRING,

    category STRING, price DOUBLE, page_source STRING, stay_time INT

-- 练习：查询前10条数据
SELECT * FROM user_behavior LIMIT 10;
```

## 练习3：大数据分析（第三周前半）

在已部署的平台上做数据分析。

### 3.1 用户行为分析

```sql
-- 练习1：各行为类型占比
SELECT event_type, COUNT(*) as cnt,
       ROUND(COUNT(*)*100.0/(SELECT COUNT(*) FROM user_behavior), 2) as pct
FROM user_behavior GROUP BY event_type ORDER BY cnt DESC;

-- 练习2：各品类浏览量和购买量
SELECT category,
       SUM(CASE WHEN event_type='view' THEN 1 ELSE 0 END) as views,
       SUM(CASE WHEN event_type='buy' THEN 1 ELSE 0 END) as buys
FROM user_behavior GROUP BY category ORDER BY buys DESC;
```

### 3.2 转化率分析

```sql
-- 练习3：漏斗分析
SELECT '浏览' as step, COUNT(DISTINCT user_id) as users FROM user_behavior WHERE event_type='view'
UNION ALL
SELECT '加购', COUNT(DISTINCT user_id) FROM user_behavior WHERE event_type='add_cart'
UNION ALL
SELECT '购买', COUNT(DISTINCT user_id) FROM user_behavior WHERE event_type='buy';
```

## 练习4：数据可视化（第三周后半）

将分析结果可视化。

### 4.1 从 Hive 导出数据

```bash
# 导出事件类型统计
hive -e "SELECT event_type, COUNT(*) FROM ecommerce.user_behavior GROUP BY event_type" \
  | sed 's/\t/,/g' > /tmp/event_stats.csv
```

### 4.2 Python 画图

```python
import pandas as pd
import matplotlib.pyplot as plt

plt.rcParams['font.sans-serif'] = ['SimHei']
plt.rcParams['axes.unicode_minus'] = False

df = pd.read_csv('/tmp/event_stats.csv', names=['type', 'cnt'])

plt.figure(figsize=(10, 6))
plt.bar(df['type'], df['cnt'], color='steelblue')
plt.title('用户行为分布')
plt.xlabel('行为类型')
plt.ylabel('次数')
plt.savefig('behavior_chart.png', dpi=150)
plt.show()
```
