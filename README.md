## 🚀 快速开始

### Pandas数据处理学习路径

1. **安装Python和Pandas**
```bash
pip install pandas
```

2. **打开Pandas数据处理实战手册.pdf**，从第1章开始循序渐进学习

3. **使用练习数据**
   - `01_电商订单数据_练习.csv` - 综合练习
   - `02_传感器数据_练习.csv` - 时间序列处理
   - `03_缺失值处理_练习.csv` - 缺失值专项
   - `04_数据格式清洗_练习.csv` - 格式清洗专项

### 大数据平台部署学习路径

1. **准备环境**
   - 准备3台Linux服务器（或虚拟机）
   - 配置好网络和主机名

2. **按照手册顺序部署**
   - 先部署Hadoop基础环境
   - 再部署Zookeeper协调服务
   - 然后部署HBase、Hive、Spark等组件
   - 最后部署Kafka、Flume、ClickHouse

3. **进行联调测试**
   - 按照第11章进行组件联调
   - 完成第12章的实战场景

## 💡 学习建议

### Pandas学习建议

1. **循序渐进**：从第1章Python基础开始，不要跳过
2. **动手实践**：每看完一章，完成对应的动手练习
3. **边学边练**：使用提供的CSV文件，跟着手册一起操作
4. **理解原理**：不要只记代码，要理解每个知识点框里的概念
5. **完成模拟题**：最后完成两道竞赛模拟题检验学习成果

### 大数据平台部署学习建议

1. **循序渐进**：按照章节顺序逐步部署，确保每个组件正常运行后再进行下一步
2. **理解原理**：不要只记命令，要理解每个组件的作用和工作原理
3. **动手实践**：多练习各种操作命令，熟悉各个组件的使用
4. **查看日志**：遇到问题时首先查看日志文件，定位问题原因
5. **联调测试**：完成部署后一定要进行数据流转测试，确保各组件协同工作
