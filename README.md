## 学习路线（30天弹性计划）

> **前提**：已学完 Python 基础（会写 if/for/def），了解 Linux 基本命令（cd/ls/vi）

### 第 1-5 天：Python 数据处理
- 📘 [Pandas数据处理实战手册](./Pandas数据处理/)
- 配套练习数据（4个 CSV 文件）
- **目标**：能独立用 Pandas 做数据清洗和统计
- **时间**：每天 2-3 小时，5 天足够

### 第 6-18 天：大数据平台部署（重点！留足 buffer）
- 📘 [大数据平台部署实战手册](./大数据平台部署/)
- **目标**：在一主两从集群上部署 9 大组件
- **时间**：至少 10 天，前 3 天通读手册，后 7 天反复练习
- **关键**：第一次部署可能需要 2-3 天调试，不要急
- 涵盖：Hadoop/HDFS/Zookeeper/HBase/Hive/Spark/Kafka/Flume/ClickHouse

### 第 19-25 天：数据分析 + 可视化
- 📘 [大数据分析实战手册](./大数据分析/)
- 📘 [大数据可视化实战手册](./大数据可视化/)
- **目标**：在平台上完成 8 个分析场景 + 画出报告图表
- **时间**：每天 2-3 小时

### 第 26-30 天：模拟竞赛
- 使用 [云端数据仿真平台](http://163.7.1.176:8889) 做完整练习
- 计时完成：部署（30分钟）→ 分析（60分钟）→ 可视化（30分钟）→ 报告（45分钟）
- 目标：3 小时内完成全流程

## 云端数据仿真平台

平台地址：**http://163.7.1.176:8889**

平台持续生成电商用户行为数据，是所有练习的数据源。

| API | 说明 |
|:---|:---|
| `/api/data?n=10` | 获取 N 条实时数据 |
| `/api/log?lines=10` | 模拟日志输出 |
| `/api/history` | 历史数据查询 |
| `/api/users` | 用户画像（1000人） |
| `/api/products` | 商品数据（500个） |
| `/api/stats` | 统计汇总 |
