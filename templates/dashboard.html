API 接口文档（共 14 个接口）

所有接口无需鉴权，直接访问。地址：http://163.7.1.176:8889

行为数据接口

GET /api/data?n=10 生成 N 条干净的实时行为数据（1-100）
GET /api/dirty?n=50 生成含缺失值/重复值/异常值的脏数据（1-200）
GET /api/dirty/csv?n=100 生成脏数据 CSV 格式（Tab 逗号分隔）
GET /api/dirty/tsv?n=100 生成脏数据 TSV 格式（Tab 分隔）
GET /api/dirty/enhanced?n=50 增强版脏数据（混合编码/日期不一致/多余空格）
GET /api/log?lines=10 逐行 JSON 日志输出（适合 Flume 采集，1-50）

数据查询接口

GET /api/history?page=1&per_page=50 历史行为数据分页查询
GET /api/history?event_type=buy 按事件类型筛选（view/click/add_cart/buy/search/collect/share）
GET /api/history?category=手机数码 按品类筛选（15个品类）

用户与商品接口

GET /api/users?page=1&per_page=100 用户画像数据（1000个用户：name/age/gender/city/member_level）
GET /api/products?page=1&category=手机数码 商品数据（500个：name/category/price/rating/sales）
GET /api/join?n=100 三表关联数据（行为+用户+商品，可用于多表 JOIN 练习）

分析与导出接口

GET /api/stats 统计汇总（事件分布/品类统计/转化率）
GET /api/timeseries?days=7&granularity=hour 按小时/天的统计数据（hour 或 day）
GET /api/export/csv?n=5000 批量导出 CSV（最多 10000 条，适合上传 HDFS）
GET /api/hive/ddl?table=user_behavior 自动生成 Hive 建表 SQL（JSON/CSV/TSV 三种格式）

Python 使用示例

import requests
import pandas as pd

# 获取干净数据
df = pd.DataFrame(requests.get("http://163.7.1.176:8889/api/data?n=500").json())

# 获取脏数据（用于清洗练习）
dirty = pd.DataFrame(requests.get("http://163.7.1.176:8889/api/dirty?n=200").json())

# 获取三表关联数据
data = requests.get("http://163.7.1.176:8889/api/join?n=100").json()
behaviors = pd.DataFrame(data['behaviors'])
users = pd.DataFrame(data['users'])
products = pd.DataFrame(data['products'])

# 批量导出 CSV（适合上传 HDFS）
csv_text = requests.get("http://163.7.1.176:8889/api/export/csv?n=5000").text

# 获取时间序列数据
ts = pd.DataFrame(requests.get("http://163.7.1.176:8889/api/timeseries?days=7").json())

# 获取 Hive 建表 SQL
ddl = requests.get("http://163.7.1.176:8889/api/hive/ddl").json()
print(ddl['create_table'])
